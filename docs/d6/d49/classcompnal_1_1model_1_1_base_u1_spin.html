Detailed Description

template<typename RealType>
class compnal::model::BaseU1Spin< RealType >

The base class for spin systems with the U(1) symmetry. Conserved quantity is the total sz \( \langle\hat{S}^{z}_{\rm tot}\rangle=\sum^{N}_{i=1}\langle\hat{S}^{z}_{i}\rangle \).
